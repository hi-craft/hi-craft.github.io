### 20.tlp 电源管理工具

​	下载方式：

```shell
sudo add-apt-repository ppa：linrunner / tlp 
sudo apt-get update
sudo apt-get install tlp tlp-rdw
```

​	启动方式：

```shell
sudo tlp start

//添加开机自启动

sudo gedit /etc/rc.local
//在exit 0前加入以下代码:
modprobe acpi_call
tlp start
```
